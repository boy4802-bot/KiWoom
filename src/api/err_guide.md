# err.py 해설서

## 이 파일은 무엇을 하나요?

`err.py`는 키움 API 호출에서 공통으로 사용하는 예외 타입을 정의합니다.
모든 모듈이 같은 예외 형식을 쓰도록 맞추기 위한 파일입니다.

## 예외 종류

- `ApiErr`
  - 키움 응답의 `return_code != 0`일 때 사용
- `HttpErr`
  - HTTP 상태코드 오류(4xx, 5xx)일 때 사용

## 보조 함수

- `ensure_ok(api_id, code, msg)`
  - code가 0이 아니면 `ApiErr`를 발생

## 왜 필요한가요?

각 파일마다 제각각 `RuntimeError`를 던지면 운영/로그 분석이 어렵습니다.
예외를 표준화하면 에러 처리 정책(재시도, 알림, 중지)을 한 번에 적용하기 쉽습니다.
